Conhecimentos

HTML

Tenho domínio dos conceitos de HTML , a cerca do funcionamento e sintaxes. Também prezo por boas praticas da linguagem de marcação, possuo conhecimentos comuns importantes a respeito de atributos, Elementos e Tags. Além disso, é importante saber relacionar de forma prática e eficaz com CSS e JavaScript.

CSS

É extremamente importante dominar este mecanismo de estilo.Tenho conhecimentos satisfatórios de formatação de estilo inline e externo. Alguns recursos como display Flex, felxbox e grid estão sempre nas minhas materias de estudo para aperfeiçoamento.

Wordpress

Tenho habilidades importantes para desenvolver interfaces e de sites através do wordpress. Me sinto capaz de compor equipe de projetos , tanto no gerenciamento quanto na implementação dos projetos.Sempre busco uma base segundo critérios de funcionalidade e estética para meus projetos. Em termos de UX e UI tenho conhecimentos básicos.

JavaScript

Apesar de não atuar ainda como Fullstack, também tenho conhecimentos de JavaScript.Meus conhecimentos e experiencias em JS estão de basico para intermediário, com experiencias em pequenos prjetos. Faço estudos da linguagem a medida em que meus projetos exigem aprofundamento e atualização.

Bootstrap

Com o ritmo de desenvolvimento solicitado hoje pelas empresas , é indispensável conhecer e saber usar o Bootstrap. Ajuda muito na produtividade, porém é interessante que não fiquemos refem da ferramenta e possamos "caminhar com as proprias pernas".

Git e GitHub

A exemplo do Bootstrap, quando falamos em produtividade e engajamento, devemos lembrar sempre do Git e GitHub.O profissional de desenvolvimento hoje precisa quase que obrigatoriamente saber utilizar essas ferramentas.
